Participé fue en la creación de un videojuego con Python. Este es uno de los proyectos que considero más importantes pues a parte de que fue un gran desafio plantear un método autodidacta de como resolver problemas de progamacíón en el menor tiempo posible para obtener avances continuos y reducir errores en el prototipado, descubrí lo importante que es el trabajo en equipo, lo que es trabajar bajo presión y como apoyarse sobre las habilidades de los demás, para poder complementarnos, y juntos encontrar un equilibrio en el trabajo. Este proyecto lo realicé con dos muy buenos amigos, quienes conocí más durante la creación del proyecto, y donde a día de hoy no me arrepiento de ello.
Implementé el algoritmo del funcionamiento de un árbol cuaternario el cual se visualizaba por medio de consola. Esta implementación se hizo del lenguaje de JavaScript a C++.
Realicé la implementación de una base de datos sobre accidentes aereos, con el fin de obtener datos sobre los accidentes aereos ocurridos hasta 2008. La base de datos relacional, almacenada en el motor de Postgresql, fue realizada para poder ser conectada a traves del módulo de psycopg2, con el fin de visualizar y usar los datos de la base de datos en una pagina web creada con el módulo de flask y Bootstrap. Añadiendo a lo anterior, participe nuevamente en la consulta de datos, sin embargo, en esta ocasión, para poder obtener datos necesarios para poder realizar pruebas de hipótesis.
Participé en la creación de un sistema domótico integrado en una casa para personas en condición de discapacidad. En este proyecto, desarrollaba el rol del prototipado y creación de la casa a escala, así como la conexion todos los elementos fisicos al Arduino Uno y al Esp-32.
Para finalizar, desarrollé una aplicación para poder visualizar como funciona el proceso de Link-State, donde por medio de la creación de una red aleatoria de nodos conectados entre si, con pesos aleatorios, los cuales cumplian la función de routers, se calculaba la mejor ruta entre dos nodos con la finalidad de poder encontrar un mejor camino en la transferencia de paquetes.
